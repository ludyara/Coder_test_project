### ***Тестовая программа для модификации (шифрования) входных файлов бинарной операцией с заданным значением (по сути инструмент для обфускации/деобфускации).***

**Desktop-приложение на C++/Qt для побайтовой модификации файлов с использованием XOR-ключа.** Программа предназначена для обработки больших объёмов данных с сохранением возможности возобновления прерванной работы за счёт многопоточности.

*   **Побайтовый XOR**: Модификация файлов с использованием 8-байтного ключа в HEX-формате (например, `A1B2C3D4E5F67890`).
*   **Обработка больших файлов**: Оптимизирована для работы с файлами от 1 МБ до 10+ ГБ без зависаний интерфейса.
*   **Многопоточность**: Интерфейс остается отзывчивым во время длительных операций ввода-вывода.
*   **Рекурсивный обход папок**: Обработка файлов во всех вложенных директориях с сохранением структуры.
*   **Режимы работы**:
    *   *Разовый* — однократная обработка выбранных файлов.
    *   *Реального времени* — периодическая проверка папок с заданным интервалом.
*   **Возобновление работы**: Если процесс прерван (кнопка "Стоп"), программа запоминает позицию в файле и продолжает обработку с того же места при следующем запуске.
*   **Конфликты имен**: Автоматическое переименование файлов (добавление суффиксов `_2`, `_3` и т.д.) или перезапись существующих.
*   **Удаление исходников**: Опциональное удаление исходных файлов после успешной обработки.
*   **Прогресс-бар и статус обработки**: Отображение статуса обработки и прогресса выполнения.

### Требования
*   **Qt Creator**: Версия 6.0 или выше (использовалась 20.0.0)
*   **Компилятор**: MinGW
*   **CMake**
